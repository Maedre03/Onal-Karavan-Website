## 🚀 The Flagship Experience: Version 3 (V3)
Welcome to the latest, fully optimized, and premium iteration of the **Önal Karavan** website. 

Version 3 represents our proudest frontend leap. We engineered this project as a **zero-dependency "Premium Vanilla" build** (HTML5, CSS3, JS). This means absolutely no bulky frameworks or bloated UI libraries—just meticulously crafted, ultra-fast native web code delivering a tier-1 agency aesthetic.

## ✨ Key Features of V3
- **Smart Hide-on-Scroll Navbar:** An intelligent, performance-optimized navigation bar that dynamically hides on scroll-down to maximize viewing area and seamlessly reappears on scroll-up.
- **Mobile-First Native UI:** Organically designed to feel like a native mobile application, providing fluid, responsive scaling from smart devices to ultra-wide desktop monitors.
- **CSS Grid Layouts:** Modern structural logic deployed for complex, pixel-perfect UI blocks within the gallery, services, and FAQ sections.
- **Accessible SVG Icons:** Razor-sharp, infinitely scalable, and ultra-lightweight graphics without relying on heavy third-party icon fonts.
- **Extreme Performance Optimizations:** Native lazy-loading, clean DOM trees, and silky-smooth 60fps hardware-accelerated CSS micro-animations.

## 🛠 Setup & Run
Because this is a pure Vanilla build, the experience is frictionless. There are **no build tools, no complex compilers, and no `node_modules`.**

To run the project locally:
1. Clone this repository to your local workspace.
2. Navigate into the `Onal_KaravanV3` directory.
3. Simply double-click `index.html` to open it in any modern web browser.

*(Pro Tip: For live-reload during local development, use an extension like VSCode's 'Live Server').*
